# Amazon Landing Page Clone

A simple **Amazon landing page clone** built using **HTML and CSS**. This project replicates the layout and basic design of the Amazon homepage for learning and practice purposes.

## 🚀 Features

* Amazon-style navigation bar
* Hero/banner section
* Product category boxes
* Grid-based layout
* Footer with multiple sections
* Responsive design using CSS

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid

## 📁 Project Structure

```
Amazon-Landing-Page-Clone/
│
├── amazon.html        # Main HTML file
├── style.css          # Styling for the webpage
├── images/            # Images used in the project
└── screenshots/       # Project preview screenshots
```
